### **步骤 1：金融新闻提取**

**脚本**：[extract_finance_news.py](Pilot/data/finance_dataset/extract_finance_news.py)

**功能**：从通用假新闻数据集中提取包含金融关键词的新闻

**核心逻辑**：
```python
# 读取金融关键词
finance_words = ['GDP', 'EBITDA', 'ROI', 'Bond', ...]

# 构建正则表达式模式
pattern = 'GDP|EBITDA|ROI|Bond|...'

# 筛选包含金融关键词的新闻
finance_articles = df[df['text'].str.contains(pattern, case=False)]
```

**处理步骤**：
1. 📥 读取原始数据集：`../archive/DataSet_Misinfo_TRUE.csv`
2. 📖 加载金融词汇表：`finance_words.txt`
3. 🔍 使用正则表达式匹配：不区分大小写
4. 🎯 筛选包含金融术语的新闻
5. 💾 输出到：`Finance_Related_Articles_TRUE.csv`

**使用方法**：
```powershell

..\..\..\..venv\Scripts\python.exe extract_finance_news.py
```

**输出示例**：
```

Finance words escaped successfully.

Found 62163 finance-related articles.
Finance-related articles saved successfully.
```

**注意事项**：
- ⚠️ 需要原始数据集位于 `../archive/` 目录
- ⚠️ 处理大文件需要较多内存（> 2GB）
- 💡 可调整 `finance_words.txt` 来定制筛选标准

**处理步骤**：

## 🔬 数据解读与应用

### **如何判断新闻真假**

模型通过以下特征判断：

#### ✅ **真实新闻特征**
- 📰 客观陈述事实
- 📊 引用具体数据和来源
- 🎯 语言中性、专业术语使用准确
- 📌 新闻结构完整（5W1H）

#### ❌ **虚假新闻特征**
- 🔴 情绪化词汇：disgusting、frightening、vile
- 🔴 极端表述：autocrat、propaganda、Hitler
- 🔴 缺乏数据支撑
- 🔴 煽动性语言：ask、stroke ego、kiss ass

### **模型评估指标**

由于数据不平衡（真:假 = 1:1.85），应重点关注：

- **Precision（精确率）**：预测为假新闻中真正是假新闻的比例
- **Recall（召回率）**：所有假新闻中被正确识别的比例
- **F1-Score**：精确率和召回率的调和平均
- **Confusion Matrix（混淆矩阵）**：详细分类结果

**不要只看 Accuracy**！在不平衡数据集上，高准确率可能只是模型偏向多数类。

## 💡 **数据使用建议**

基于实际数据分析，以下是最佳实践：

### ✅ **训练策略**
1. **特征工程**：
   - 提取情感词密度作为特征
   - 计算金融术语覆盖率
   - 使用 TF-IDF 捕捉词频特征
   - 添加文本长度、标点符号密度等

2. **模型选择**：
   - **朴素贝叶斯**：快速原型，适合线上实时预测
   - **CNN**：捕捉局部特征和语义模式
   - **Pilot 模型**：针对金融场景优化

3. **验证方法**：
   - 使用 **Stratified K-Fold** 保持类别分布
   - 重点关注 **Precision**（避免误判真新闻为假）
   - 关注 **Recall**（尽可能发现所有假新闻）

### ⚠️ **注意事项**
- 数据集主要是政治类新闻（Trump、Clinton 高频出现）
- 金融关键词密度较低，需要领域适配
- 情感词是最强特征，但可能过拟合
