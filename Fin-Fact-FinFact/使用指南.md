# 📖 Fin-Fact-FinFact 使用指南

> 多模态金融事实核查数据集 - 完整使用文档

## 📌 项目概述

**Fin-Fact-FinFact** 是一个专注于金融领域的多模态事实核查数据集，结合了文本声明和相关图片信息，用于训练和评估自动事实核查模型。

### 🎯 核心特点

- 🌐 **权威数据源**：数据来自 [Snopes.com](https://www.snopes.com) 和 [PolitiFact.com](https://www.politifact.com) 两大专业事实核查网站
- 💼 **金融专业性**：聚焦经济、税收、预算、投资等金融主题
- 🖼️ **多模态支持**：75.1% 的记录包含图片信息，支持视觉与文本结合的事实核查
- 📊 **丰富标注**：包含声明、证据链、详细解释、主题标签等多维度信息
- 🔬 **研究就绪**：提供多种预训练模型评估脚本（BERT、BART、GPT、Gemini 等）

### 🏷️ 标签分布

```
false (假): 1,492 条 (44.3%)  ████████████████████
true (真):  1,275 条 (37.8%)  █████████████████
NEI (信息不足): 602 条 (17.9%)  ████████
```

**标签说明**：
- **true**: 声明被证实为真
- **false**: 声明被证实为假
- **NEI** (Not Enough Information): 缺乏足够证据判定真伪

⚠️ **注意**：数据集存在轻度不平衡（false > true > NEI），建议使用加权损失或数据增强技术。

```json

      "sentence": "证据句子",

    }

  "image_data": [
    {
      "image_url": "图片链接",
      "image_caption": "图片标题"
    }
  ],
  "label": "true/false/NEI"
}
```

## 🚀 快速开始

### 1️⃣ 加载数据集

```python
import json

# 读取数据集
with open('finfact.json', 'r', encoding='utf-8') as f:
    dataset = json.load(f)

print(f"总记录数: {len(dataset)}")

# 查看第一条记录
sample = dataset[0]
print(f"\n声明: {sample['claim']}")
print(f"标签: {sample['label']}")
print(f"证据数: {len(sample['evidence'])}")
print(f"主题: {', '.join(sample['issues'])}")
```

### 2️⃣ 基本数据分析

```python
from collections import Counter

# 统计标签分布
labels = [item['label'] for item in dataset]
label_counts = Counter(labels)
print("标签分布:", dict(label_counts))

# 统计数据来源
sources = [item['url'].split('/')[2] for item in dataset]
source_counts = Counter(sources)
print("数据来源:", dict(source_counts))

# 统计图片覆盖率
with_images = sum(1 for item in dataset if item.get('image_data'))
print(f"包含图片: {with_images}/{len(dataset)} ({with_images/len(dataset)*100:.1f}%)")
```

### 3️⃣ 运行数据分析脚本

```bash
# 使用提供的分析脚本生成详细报告
python analyze_finfact.py
```

该脚本会自动生成：
- 📊 标签和来源分布统计
- 📏 文本长度分布分析
- 🔤 高频词汇提取
- 🖼️ 图片覆盖率统计
- 📝 数据样本展示

## 🔬 模型评估

### 📁 Scripts 目录结构

```
scripts/
├── data_scrapper.py           # 爬虫基类
├── data_crawler_snopes.py     # Snopes 爬虫
├── data_crawler_politifact.py # PolitiFact 爬虫
├── data_generator_snopes.py   # Snopes 数据生成器
├── data_generator_politifact.py # PolitiFact 数据生成器
└── models/
    ├── anli.py                # ANLI 模型评估（BART/RoBERTa/ELECTRA/AlBERT/XLNET）
    ├── gpt2_nli.py            # GPT-2 事实核查
    ├── bart_eval.py           # BART 模型评估
    ├── bert_gen.py            # BERT 生成模型
    ├── experiments.py         # 多模态 LLM 实验（LLaVA/GPT-4/Gemini）
    ├── prompts.py             # LLM 提示模板
    ├── bart_explaination_gen.py  # BART 解释生成
    ├── pegasus_gen.py         # PEGASUS 模型
    ├── xl_sum_gen.py          # XL-Sum 模型
    ├── llm_gemini.py          # Gemini 文本模型
    ├── lvlm_gemini.py         # Gemini 视觉模型
    ├── lvlm_gpt_vision.py     # GPT-4 Vision
    └── get_api.py             # API 密钥管理
```

### 🤖 运行基础模型（无需 API）

以下模型可直接使用，无需 API 密钥：

```bash

cd scripts/models
python anli.py

# GPT-2 事实核查
python gpt2_nli.py

python bart_eval.py
```

## 💡 使用建议

### 🎓 学术研究

1. **基线模型对比**：使用 `anli.py` 评估多个预训练模型性能
